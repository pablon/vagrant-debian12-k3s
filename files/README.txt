 Development VM Environment

- To enter the VM run 'vagrant ssh', become root with 'sudo su -'
- To stop the VM run 'vagrant halt', to restart run 'vagrant up'
- To restore the VM snapshot run 'vagrant snapshot restore'
- To destroy the VM run 'vagrant destroy -f'
